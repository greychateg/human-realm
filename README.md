# 人间修真模拟器 🧘

> 以王阳明《传习录》为心法，以修仙境界为框架，专为交易员设计的心性修行工具。

**在线体验：** [renjianxiuzhen.netlify.app](https://renjianxiuzhen.netlify.app)

---

## 这是什么？

做交易的人都知道：真正的对手不是市场，是自己的心魔。FOMO 追高、恐惧割肉、重仓赌气、频繁交易——每一种亏损背后都有一种心理陷阱。

**人间修真模拟器**把「修炼心性」这件事游戏化了。你不是在打坐冥想，你是在**修仙**——从练气期到元婴期，每天修炼、参悟、记录心魔，一步步提升境界。内核是严肃的：底层心法全部来自王阳明《传习录》213则原文。

## 核心玩法

### ✦ 机缘 — 每日金句

从 **646 条** 精选《传习录》语录中随机抽取。每条都是 50-100 字的独立金句，配有白话解读和分类标签。每天 8 次领悟机会，获取灵力。

### ⚔ 修炼 — 七大功法

| 功法 | 类型 | 说明 |
|------|------|------|
| 🌬 吐纳术 | 调息 ★ | 4-4-6 呼吸法，90秒引导 |
| 🔥 炼心术 | 觉察 ★★ | 八区身体扫描，内观觉照 |
| ⚔ 明断术 | 果决 ★★ | 盘前推演，预判FOMO |
| 💎 固本术 | 接纳 ★ | 正向复盘，感恩练习 |
| 🎯 凝神术 | 入定 ★★★ | 25分钟深度专注计时 |
| 🌙 省察术 | 暮省 ★★ | 日终复盘 + 五维评分 |
| 📜 格物术 | 知行 ★★★ | 念头溯源，知行合一对照 |

重复练习有衰减机制——鼓励你每天尝试不同功法，而非只练最熟悉的。

### 🔥 入魔 — 心魔记过

**11 种**详细交易心魔场景，按「财·色·名·利」四类分类：

- 追涨杀跌、FOMO 加仓、重仓赌气、频繁交易……
- 每种心魔有具体场景描述和魔气惩罚值
- 支持 **AI 智能识别**：用自己的话描述发生了什么，自动匹配心魔类型
- 入魔后推荐对治心法（关联参悟篇目）

### 📖 参悟 — 传习录深读

**213 则**《传习录》完整收录，支持：

- 按标签筛选（致良知、知行合一、克己、静坐……）
- 原文 + AI 白话解读
- 参悟计数，重复研读有加成
- TTS 语音朗读

### 🏆 除妖试炼

高难度心性挑战，完成获丰厚灵力并降低魔气。

## 修真体系

```
练气期 → 筑基期 → 金丹期 → 元婴期
  初期 → 中期 → 后期 → 大圆满
```

- **灵力（XP）**：修炼、参悟、机缘获得，升级突破境界
- **魔气**：72小时不修炼则魔气满值，降低修炼效率（灵力倍率下降）
- **连续打卡**：连续天数越多，灵力倍率越高
- **境界突破**：升级时触发动画 + 铜钟音效

## 技术特点

- **纯前端单文件** — 一个 `index.html` 搞定一切，零依赖，无需构建
- **PWA 支持** — 可添加到手机桌面，离线可用
- **数据本地化** — 全部存储在 `localStorage`，支持导出/导入 JSON
- **响应式设计** — 移动端优先，适配 iPhone 安全区域
- **音效系统** — 铜钟、点击音效、环境音乐、TTS 语音
- **触觉反馈** — 移动端全局振动反馈
- **AI 集成** — 参悟解读 + 入魔识别（Anthropic Claude API，需自行配置）

## 快速开始

### 本地使用

```bash
git clone https://github.com/yourname/renjian-xiuzhen.git
open index.html
```

浏览器直接打开即可使用，无需任何安装。

### 部署到 Netlify / Vercel / GitHub Pages

直接部署整个仓库，无需构建步骤。

### 配置 AI 功能（可选）

在代码中找到 `ANTHROPIC_KEY` 变量，填入你的 API Key 即可启用 AI 参悟解读和智能入魔识别。没有 Key 也不影响其他功能使用。
